Detailed Description

The Game class is responsible for the current game turn being played. It is dervived from the GameState class, meaning that it is one of the possible states that the video game can be in at any given time. The state that this class represents is that state of the game while the user is playing a turn. It is responsible for drawing and updated the window during the game turn and handling any events that it recieves.
